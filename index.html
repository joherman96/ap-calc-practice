Warm-Up 1 Disk
The region bounded by y = x², y = 0, and x = 2 is rotated about the x-axis. Find the volume of the resulting solid.
Work it out here or on paper...
V = 32π/5
Disk method — the region touches the x-axis, so no inner radius.

V = π ∫02 (x²)² dx = π ∫02 x4 dx

= π [x5/5]02 = π(32/5) = 32π/5
Warm-Up 2 Washer
The region bounded by y = x and y = x² (from x = 0 to x = 1) is rotated about the x-axis. Find the volume.
Work it out here or on paper...
V = 2π/15
Washer method — gap between y = x² and the outer curve y = x.

Outer radius R = x,   Inner radius r = x²

V = π ∫01 [x² − x4] dx = π [x³/3 − x5/5]01

= π(1/3 − 1/5) = π(2/15) = 2π/15
Part A: Multiple Choice (10 Questions)
1. Inverse Trig Derivative
Find d/dx [tan−1(3x)].
(A)   1/(1 + 9x²)
(B)   3/(1 + 9x²)
(C)   3/(1 + 3x²)
(D)   −3/(1 + 9x²)
(E)   1/(1 + 3x²)
Answer: (B)
d/dx [tan−1(3x)] = 1/(1 + (3x)²) · 3 = 3/(1 + 9x²)
Don't forget the chain rule — multiply by the derivative of the inside.

(D)   tan−1(x/3) + C
(E)   cos−1(x/3) + C

Here a = 3, so: sin−1(x/3) + C

7. U-Sub + Inverse Trig
Evaluate ∫ 6x/(1 + 9x4) dx.

(B)   (1/3)tan−1(3x²) + C
(C)   3tan−1(x²) + C
(D)   sin−1(3x²) + C

Answer: (A)
Let u = 3x², then du = 6x dx.

8. Product Rule + Inverse Trig
Find d/dx [x · sin−1(x)].
(A)   x/√(1 − x²)
(B)   sin−1(x) + x/√(1 − x²)
(C)   sin−1(x) − x/√(1 − x²)
(D)   1/√(1 − x²)
(E)   sin−1(x) + 1/√(1 − x²)
Answer: (B)
Product rule: (x)' · sin−1(x) + x · (sin−1(x))'
= 1 · sin−1(x) + x · 1/√(1 − x²)
= sin−1(x) + x/√(1 − x²)
9. Inverse Trig Integral
Evaluate ∫ dx/√(5 − x²).
(A)   sin−1(x/5) + C
(B)   sin−1(x/√5) + C
(C)   (1/√5)sin−1(x) + C
(D)   tan−1(x/√5) + C
(E)   √5 · sin−1(x) + C
Answer: (B)
∫ dx/√(a² − x²) = sin−1(x/a) + C.
Here a² = 5 so a = √5: sin−1(x/√5) + C
Common mistake: using a² instead of a. It's x/√5, not x/5.

V = 15π/2

14. Washer Method
The region bounded by y = x + 2 and y = x² is rotated about the x-axis. Find the volume of the resulting solid.
Show all work...
V = 72π/5
Step 1: Find intersections. x + 2 = x² → x² − x − 2 = 0 → (x − 2)(x + 1) = 0.
x = −1 and x = 2.

Step 2: Identify radii. On [−1, 2], x + 2 > x².
Outer: R = x + 2,   Inner: r = x².

Step 3: Set up and integrate.
V = π ∫−12 [(x + 2)² − (x²)²] dx
= π ∫−12 [(x² + 4x + 4) − x4] dx
= π [x³/3 + 2x² + 4x − x5/5]−12

At x = 2: 8/3 + 8 + 8 − 32/5 = 40/15 + 240/15 − 96/15 = 184/15
At x = −1: −1/3 + 2 − 4 + 1/5 = −5/15 − 30/15 + 3/15 = −32/15

V = π(184/15 − (−32/15)) = π(216/15) = 72π/5
15. Inverse Trig Application
Let g(x) = tan−1(x² − 1).

(a) Find g'(x).
(b) Find the equation of the tangent line to y = g(x) at x = 1.
(c) Is g concave up or concave down at x = 1? Justify.
Show all work...
Answers below:
(a) g'(x) = 1/(1 + (x² − 1)²) · 2x = 2x / (1 + (x² − 1)²)
Can expand denominator: 1 + x4 − 2x² + 1 = x4 − 2x² + 2.
So g'(x) = 2x/(x4 − 2x² + 2).

(b) g(1) = tan−1(1 − 1) = tan−1(0) = 0. Point: (1, 0).
g'(1) = 2(1)/(1 − 2 + 2) = 2/1 = 2.
Tangent line: y = 2(x − 1)   or   y = 2x − 2

(c) Need g''(1). Using quotient rule on g'(x) = 2x/(x4 − 2x² + 2):
Numerator of g'': 2(x4 − 2x² + 2) − 2x(4x³ − 4x) = 2x4 − 4x² + 4 − 8x4 + 8x² = −6x4 + 4x² + 4.
At x = 1: −6 + 4 + 4 = 2 > 0. Denominator is positive.
g''(1) > 0, so g is concave up at x = 1.
